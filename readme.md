## 特性

1. 命令式浏览和下载预设，直接下载到 ChatLuna 的预设文件夹
2. 其他特性正在制作中....

**不是无缝集成，下载完后需要手动切换**

## 部署

在 Koishi 插件市场搜索 `chatluna-preset-market`，安装后启用即可。

**插件依赖 ChatLuna，请确保在 ChatLuna 启动后再启用本插件。**
